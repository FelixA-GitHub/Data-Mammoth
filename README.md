## Data Mammoth

# Description
- A simple search engine for movies, with a favorites list

# User Story
- AS A movie fan
- I WANT search for movies
- SO THAT I can see recommendations, information, and add to a favorites list

# Technologies Used
- CSS
- HTML
- JavaScript
- jQuery
- Materialize.css
- Web APIs
- Modals
- GitHub Pages
